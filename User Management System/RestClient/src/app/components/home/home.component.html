User Management System Application Angular15+Material UI

Technology:

  1. Web Portal: Angular
  2. BackEnd: NodeJs
  3. Database: PostgreSQL

Features:

  • User can Register themselves
  • Without Registration user cann't be LoggedIn
  • Forget Password feature available in Login session
  • User can update his/her profile and can show other users in a tabular form
  • User can update his/her profile picture
  • user can logout from his profile
